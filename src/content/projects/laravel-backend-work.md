The `mbsoft31` account is the older personal and developer account. It now works as a supporting proof surface for Laravel/PHP backend and package work, while the current research-tooling flagship lives under `nexus-scholar`.

## Best-Fit Paid Work

- Laravel/PHP backend contracts.
- API design, integrations, queues, admin workflows, and data modeling.
- SaaS MVP foundations where the backend domain is more important than decoration.
- Package extraction and codebase cleanup.
- Research or education software where auditability and data flow matter.

## Public Proof

The strongest non-Nexus pinned proof is `loyalty-rewards`, a PHP package around rewards-domain modeling: points, redemption, fraud checks, audit logging, and multi-tier reward programs.

`quransearch` is a different kind of proof: a Go text-search utility over local Quran text data, with exact and pattern-based search. It is not the main Laravel story, but it shows careful command-line/data work.

Other repos such as `store`, `databank`, `math-school`, `questionbank`, `deploy.mouadh.tech`, and the older systematic-review prototypes are useful when they support a specific article, demo, or contract conversation. They should not all compete for the front page at once.

## Public Strategy

- Nexus Scholar for current research software direction;
- selected Laravel/PHP packages for backend credibility;
- older prototypes available only when they support a specific story;
- this site itself as a live deployment artifact: Astro, Docker Compose, Nginx, Caddy, and Let's Encrypt on a VPS.

# mouadh.org

Personal technical site for Mouadh Bekhouche.

The site is Astro, Markdown-first, and evidence-first. It is designed to be the canonical public archive for senior Laravel/PHP backend work, Nexus Scholar, research workflow infrastructure, and carefully bounded research writing.

## Commands

```bash
npm install
npm run dev
npm run build
```

Canonical public domain: `https://mouadh.org`.

## Content Model

- `src/content/writing`: public essays and technical posts.
- `src/content/projects`: project pages.
- `src/content/evidence`: evidence packets and source notes that support public claims.

Prefer Markdown for ordinary posts. Use MDX only when a post needs interactive or reusable components.

## Public Boundary

Public:

- Nexus Scholar architecture and package surfaces.
- Laravel/PHP backend engineering notes.
- Evidence-backed research workflow writing.
- Plant-disease AI direction with benchmark-safe wording.

Private:

- credentials and deployment secrets;
- client data;
- unpublished thesis methods;
- private roadmap details;
- unclear-license raw datasets.
